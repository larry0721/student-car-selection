# components

Reusable React components for the advisor interface.

- `BuyerProfilePlanner.tsx`: questionnaire, scoring-weight editor, state management, and top-ten recommendation layout.
- `DataImportPanel.tsx`: optional online enrichment and CSV score import controls.
- `RecommendationCard.tsx`: individual recommendation report card with weighted score breakdown.

Keep components focused on presentation and light interaction. Shared scoring or cost logic belongs in `lib/`.
